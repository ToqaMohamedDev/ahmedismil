# موقع أحمد إسماعيل (المشير)

موقع تعليمي لـ **أحمد إسماعيل** - مدرس تاريخ وجغرافيا، لقب **المشير**.

## المميزات

- ✅ Next.js 14 مع App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Firebase Authentication (Google + Email/Password)
- ✅ Firestore لتخزين البيانات
- ✅ Dark Mode
- ✅ Framer Motion للأنيميشن
- ✅ تصميم متجاوب (Responsive)

## التثبيت

1. تثبيت المتطلبات:
```bash
npm install
```

2. إعداد Firebase:
   - أنشئ مشروع Firebase جديد
   - انسخ ملف `.env.example` إلى `.env.local`
   - أضف بيانات Firebase من Console

3. تشغيل المشروع:
```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

## الصفحات

- `/` - الصفحة الرئيسية
- `/videos` - الفيديوهات التعليمية
- `/courses` - الكورسات
- `/tests` - الاختبارات التفاعلية
- `/login` - تسجيل الدخول
- `/signup` - إنشاء حساب جديد

## البناء

```bash
npm run build
npm start
```
