# دليل المساهمة

شكراً لاهتمامك بالمساهمة في مشروع المشير!

## البدء

1. Fork المشروع
2. أنشئ branch جديد (`git checkout -b feature/AmazingFeature`)
3. Commit التغييرات (`git commit -m 'Add some AmazingFeature'`)
4. Push للـ branch (`git push origin feature/AmazingFeature`)
5. افتح Pull Request

## معايير الكود

- استخدم TypeScript
- اتبع ESLint rules
- اكتب تعليقات واضحة
- استخدم أسماء متغيرات وصفية

## هيكل المشروع

```
app/              # صفحات Next.js
components/        # مكونات React
lib/              # Utilities و helpers
types/            # TypeScript types
public/           # ملفات ثابتة
```

## الإبلاغ عن المشاكل

عند الإبلاغ عن مشكلة، يرجى تضمين:
- وصف واضح للمشكلة
- خطوات إعادة إنتاج المشكلة
- المتوقع مقابل ما يحدث فعلياً
- معلومات البيئة (OS, Browser, Node version)

## اقتراح ميزات

نرحب بجميع الاقتراحات! يرجى فتح issue جديد مع:
- وصف واضح للميزة
- سبب الحاجة إليها
- أمثلة على الاستخدام
